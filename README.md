# 🦠 세균전 게임

초등학생을 위한 귀여운 세균전 모바일 게임입니다!

## 🎮 게임 방법

### 목표
상대방보다 많은 귀여운 세균 친구들을 만들어 승리하세요!

### 규칙

1. **말 놓기**: 자신의 세균에서 한 칸 또는 두 칸 떨어진 빈 칸에 말을 놓을 수 있습니다.
   - **한 칸**: 그 자리에 새로운 세균을 추가합니다
   - **두 칸**: 기존 세균을 그 자리로 이동합니다

2. **전염**: 말을 둔 곳 주변(가로, 세로, 대각선)에 있는 상대방의 세균은 모두 내 세균으로 바뀝니다!

3. **승리**: 더 이상 둘 곳이 없으면 게임이 끝납니다. 세균이 많은 쪽이 승리합니다!

## 🚀 실행 방법

1. `index.html` 파일을 웹 브라우저로 열기
2. 또는 웹 서버에 업로드하여 실행

## 🎯 게임 모드

- **컴퓨터와 대전**: AI와 함께 즐기는 싱글 플레이어 모드
- **두 명이 함께**: 같은 모바일 기기에서 번갈아가며 플레이하는 멀티플레이어 모드
